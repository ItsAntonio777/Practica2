Conceptos Basicos de base de datos

¿Qué es una Base de Datos Relacional? Es un sistema que guarda datos en tablas y permite relacionarlos entre sí. La idea es que la información no esté duplicada, sino conectada de manera lógica. Por ejemplo: una base de datos de una tienda puede tener una tabla de Clientes y otra de Pedidos, y ambas se relacionan para saber qué cliente hizo qué pedido.
¿Qué es una Tabla? Una tabla es como una hoja de cálculo: tiene filas y columnas. Cada tabla representa un tema o entidad, como Productos, Empleados o Ventas.
¿Qué es una Relación? Es el vínculo entre dos tablas. Gracias a las relaciones, podemos conectar datos sin repetirlos. Por ejemplo, un pedido se relaciona con un cliente a través de una clave.
¿Qué representa una Columna? Cada columna es una característica de la entidad. En una tabla de Clientes, las columnas pueden ser Nombre, Correo y Teléfono. Piensa en ellas como las etiquetas que describen qué tipo de información se guarda.
¿Qué representa un Renglón? Cada renglón (o fila) es un registro único. En la tabla de Clientes, un renglón sería un cliente específico con su nombre, correo y teléfono.
¿Qué es un Motor de Base de Datos Relacional? Es el software que hace que todo funcione. Se encarga de guardar, consultar y mantener la información organizada. Sin el motor, las tablas serían solo texto sin vida.
Motores de Bases de Datos Relacionales más Populares MySQL PostgreSQL Oracle Database Microsoft SQL Server MariaDB
Motor Ligero que no necesita Servidor SQLite. Es tan sencillo que no requiere instalar un servidor, se integra directamente en las aplicaciones. Por eso es muy usado en móviles y proyectos pequeños.
¿Qué significa SQL? SQL es el lenguaje que usamos para hablar con la base de datos. Sus siglas vienen de Structured Query Language. Con SQL podemos pedir datos, agregar nuevos, modificarlos o eliminarlos.
Elementos de un Sistema de Información Relacional Base de datos: el conjunto de tablas y relaciones. Motor: el software que administra todo. SQL: el lenguaje para interactuar. Usuarios y aplicaciones: quienes consultan o modifican la información. Interfaces: herramientas que facilitan el acceso, como aplicaciones gráficas o sistemas web.
